## Assignment Overview
This repository contains my Product Manager assignment for designing an MVP e-commerce system from scratch.

The solution focuses on the core transaction flow:

Browse Products → Add to Cart / Buy Now → Submit Order → Pay → Ship → Complete Order

## Product Scope
The MVP supports the basic e-commerce transaction lifecycle, including customer product browsing, shopping cart, order submission, payment, shipment, order completion, product management, inventory management, and order management.

## Design Approach
The solution prioritizes the basic transaction flow first, using P0, P1, and P2 feature classification.

The main focus is on:

- Completing the end-to-end purchase flow
- Defining clear order and inventory status logic
- Handling common exception scenarios
- Supporting internal operations staff
- Monitoring core product and business metrics after launch
